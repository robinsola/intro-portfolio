Projects

Here are some of the projects I have been working on at Epicodus:

  • Washington Post

    • Here is an exact replica of the homepage of the Washington Post website. The purpose of this project was to combine html elements with bootstrap classes and a custom css styleheet.
    languages used: html, css, bootstrap

  • Spain Vacation

    • Here you can view photos of different cities in Spain and learn about the different things to do throughout the country. This vacation website was created to practice bootstrap elements such as using a container, a jumbotron, creating rows and columns, and using blockquotes.
    languages used: html, css, bootstrap

  • Blind Owl Band

    • This website highlights the Blind Owl Band, a bluegrass band from Saranac Lake, NY. View photos of the band and check out where they will be touring next. Creating this website was an introuction to using floats for creating sidebars and columns.
    languages used: html, css

  • Dog Life

    • This website is about my dog Pepper. Follow his life and interests, see his favorite places around Portland, find out his favorite snacks, and see how adorable he looks in his new fall fashion sweater.
    languages used: html, css

  • Animal Shelter

    • This is a mock website for a Multnomah County Animal Shelter. The content includes photos and animal profiles created to practice cascading styles and the box model.
    languages used: html, css

  • Cupcake Shop

    • On this website you can check out the most popular cupcakes, and the seasonal flavor at the Cupcake Shop. This page includes text and images that are styled with classes.
    languages: html, css
